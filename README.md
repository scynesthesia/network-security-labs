# Network Security & Analysis Labs

Documentation and findings from hands-on network security, traffic analysis, and Linux hardening exercises.

---

## Linux System Hardening

Documented a baseline Linux hardening guide for Ubuntu/Debian servers intended for lab and cloud-hosted environments.

### Areas of Focus
- SSH security hardening
- Host-based firewall configuration
- Least-privilege administration
- Patch and update management
- Brute-force mitigation
- Attack surface reduction

### Documentation
- [Linux Server Hardening Guide](./Docs/hardening-guide.md)

---

## Traffic Analysis with Wireshark

Analyzed protocol-level traffic to identify anomalies and troubleshoot connectivity issues.

### Areas of Focus
- Packet capture and protocol inspection
- DNS, DHCP, and HTTP/S traffic analysis
- Identifying abnormal or unexpected behavior
- Understanding packet-level communication flows

---

## Vulnerability Assessment with Nmap

Conducted network mapping and service discovery scans to simulate security audit activity in controlled lab environments.

### Areas of Focus
- Host discovery
- Port scanning
- Service enumeration
- Identifying exposed services and potential attack surface

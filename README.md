# Network Security & Analysis Labs

Documentation and findings from network security audits, traffic analysis, and system hardening exercises.

### Traffic Analysis with Wireshark
* Analyzed protocol-level traffic to identify anomalies and troubleshoot connectivity issues.
* Documented captures of common protocols (DNS, DHCP, HTTP/S) to understand packet-level behavior.

### Vulnerability Assessment with Nmap
* Conducted network mapping and service discovery scans to simulate security audits.
* Identified open ports and potential attack vectors in controlled lab environments.

### Linux System Hardening
* Configured Ubuntu/Debian servers as secure entry points, focusing on SSH hardening and firewall configuration.
